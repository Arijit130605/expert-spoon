Basic Formulae

    ∫ 1 dx = x + C
    ∫ a dx = ax+ C
    ∫ xn dx = ((xn+1)/(n+1))+C ; n≠1
    ∫ sin x dx = – cos x + C
    ∫ cos x dx = sin x + C
    ∫ sec2x dx = tan x + C
    ∫ csc2x dx = -cot x + C
    ∫ sec x (tan x) dx = sec x + C
    ∫ csc x ( cot x) dx = – csc x + C
    ∫ (1/x) dx = ln |x| + C
    ∫ ex dx = ex+ C
    ∫ ax dx = (ax/ln a) + C ; a>0,  a≠1

Rational Functions

    ∫ 1 dx = x + C
    ∫ a dx = ax+ C
    ∫ (1/x) dx = ln |x| + C

Integration of Trignomentric Functions

    ∫ sin x dx = – cos x + C
    ∫ cos x dx = sin x + C
    ∫ sec2x dx = tan x + C
    ∫ csc2x dx = -cot x + C
    ∫ sec x (tan x) dx = sec x + C
    ∫ csc x ( cot x) dx = – csc x + C

Exponential Functions

    ∫ xn dx = ((xn+1)/(n+1))+C ; n≠1
    ∫ ex dx = ex + C
    ∫ ax dx = (ax/ln a) + C ; a > 0,  a≠1
